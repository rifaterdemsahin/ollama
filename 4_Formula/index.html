Quick Start
MacBook step-by-step guide
  • Install Ollama
    Download and install via Homebrew or curl installer
    # via Homebrew brew install ollama # or via curl curl -fsSL https://ollama.com/install.sh | sh
  • Verify Installation
    Confirm Ollama is installed and check the version
    ollama --version
  • Start Ollama Service
    Launch the local inference server on port 11434
    ollama serve
  • Pull a Model
    Download a language model (new terminal window)
    ollama pull llama3.2 ollama pull nomic-embed-text # for embeddings
  • Test the Installation
    Run a test inference to confirm everything works
    ollama run llama3.2 "Hello, how are you?"

Common Issues
Port 11434 in use
Another process is using port 11434. Run lsof -i :11434 to find and kill it before starting Ollama.
Permission denied
Use sudo for the installation command, or ensure your user has the necessary permissions.
Not enough RAM
Minimum 8GB required. Use smaller models like phi3 or mistral on memory-constrained systems.
